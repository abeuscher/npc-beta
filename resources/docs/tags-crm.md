# CRM Tags

Tags are labels you apply to contacts to categorize and segment them. A contact can have multiple tags. Tags are the primary tool for grouping contacts for outreach, reporting, or filtering.

## Tags List

The Tags index shows all available CRM tags and how many contacts currently carry each tag.

## Creating a Tag

- **Name** (required) — a short, descriptive label. Use consistent naming conventions across your team (e.g., "Volunteer", "Major Donor", "Board Member").

## Applying Tags to Contacts

Tags are applied from the contact's edit form. Open a contact, locate the **Tags** field, and select one or more tags. Tags can also be applied in bulk from the Contacts list view.

## Tag Conventions

A few guidelines that make tags more useful:

- Keep tag names concise and unambiguous.
- Prefer a small, well-defined set of tags over many overlapping ones.
- Review your tag list periodically and merge or remove tags that are redundant.
- Use tags for stable, meaningful categories — not for temporary notes (use the Notes feature instead).
