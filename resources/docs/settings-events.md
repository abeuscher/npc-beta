# Events Settings

Events Settings controls default behaviour when new events are created. Access is restricted to super-admin users. This page is not currently shown in the navigation — the same setting is also available under **Settings → CMS**.

## New Events

- **Auto-publish new events** — when enabled, the landing page for a newly created event is set to **Published** automatically. When disabled, new events default to **Draft** and must be published manually before they appear on the public site.
