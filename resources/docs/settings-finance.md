# Finance Settings

Finance Settings is a placeholder for upcoming payment and accounting integrations. The page is accessible to super-admin users but does not yet contain configurable fields.

Stripe API keys and QuickBooks OAuth credentials are currently stored under **Settings → General → Integrations**. A dedicated Finance Settings page with full Stripe webhook configuration, default currency, and fiscal year start date will be built as part of the Finance sessions.
