# Design System

The Design System editor provides centralized control over the visual style of your public site. Changes made here apply globally to all pages and widgets.

## Buttons

Configure the appearance of the five built-in button variants used across your site:

- **Primary** — The main call-to-action button (e.g., "Donate Now", "Register")
- **Secondary** — A less prominent alternative (outlined style by default)
- **Text** — A minimal, text-only button style
- **Destructive** — Used for irreversible actions (e.g., "Cancel Registration")
- **Link** — Renders as an underlined link

For each variant you can control border-radius, colors, border, hover behavior, font weight, and text transform. Padding, sizing, and transitions are managed by the system for consistency.
