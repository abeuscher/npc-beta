## The pattern

Every resource table should follow this exact pattern. Use `CustomFieldDefResource` as the reference implementation.

**Row actions:**
```php
->actions([
    Tables\Actions\EditAction::make(),
    Tables\Actions\DeleteAction::make(),
])
```

**Bulk actions:**
```php
->bulkActions([
    Tables\Actions\BulkActionGroup::make([
        Tables\Actions\DeleteBulkAction::make(),
    ]),
])
```

The bulk action group produces a "Bulk Actions" dropdown with a checkbox column on the left of the table and a select-all checkbox in the header — this is standard Filament behaviour, no custom code required.
