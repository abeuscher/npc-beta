This is **execution-order position: emergent**, not tied to any `release-plan.md` entry. **Session shape:** housekeeping / pipeline cleanup. Three concrete fixes inside the public-render pipeline: (1) retire the per-instance `widgetType->css`/`js` inline emission in `WidgetRenderer` + the corresponding accumulators in `PageBlockRenderer` + `ChromeRenderer` + the `<style>$__allInlineStyles</style>` and `<script>$__allInlineScripts</script>` blocks in the public layout; (2) remove the dead `$widgetAssets` collection plumbing (collected in three places, consumed nowhere); (3) judgment call at session start on whether template `custom_scss` moves into the build bundle or stays inline (see open question in the session prompt). Pest regression tests on the rendered `<head>` shape are the close-gate.
