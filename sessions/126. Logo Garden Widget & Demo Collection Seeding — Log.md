## What was built

### Phase 1 — Widget type registration

- Added `logo_garden` widget type to `WidgetTypeSeeder` with config schema: `collection_handle`, `image_field`, `display_mode` (static/carousel/smooth/flipper), `show_name`, `name_field`, `background_color`, `logos_per_row`, `logo_max_height`, `carousel_duration`, `flip_duration`.
- Collections: `['logos']`. SCSS asset: `resources/scss/widgets/_logo-garden.scss`.

### Phase 2 — Blade template and SCSS

- Created `resources/views/widgets/logo-garden.blade.php` with four display modes:
  - **Static** — flexbox grid with configurable logos per row, `justify-content: center` for centered last row, responsive breakpoints.
  - **Carousel** — Swiper.js autoplay only, no user controls (`allowTouchMove: false`, no arrows), responsive breakpoints.
  - **Smooth scroll** — Swiper.js with `freeMode: true`, `autoplay: { delay: 0 }` for continuous ribbon loop. Added `FreeMode` module to `window.SwiperModules` in `resources/js/public.js`.
  - **Flipper** — CSS 3D `rotateY(180deg)` with `backface-visibility: hidden`, Alpine.js `setInterval` toggle, staggered delays per cell. Each cell shows current logo on front, next logo (wrapping) on back.
- Created `resources/scss/widgets/_logo-garden.scss` with flexbox layout, 3D flip styles, responsive breakpoints at `$bp-md` and `$bp-sm`.
- Logo sizing: both `max-height` and `max-width` constrained to the same CSS variable (`--logo-max-height`, default 150px) so logos fit in a square box regardless of aspect ratio.
- Carousel cells get `width: auto` to prevent conflict with Swiper's own slide width calculations (fixed a ~25% offset issue).

### Phase 3 — Demo collection seeder

- Created `database/seeders/LogoGardenDemoSeeder.php` with collection handle `logo-garden-demo`, fields `name` (text) and `logo` (image), 9 items with sample logo images attached via Spatie media library from `resources/sample-images/logos/`.
- Added a `logo` media conversion (200x200, webp) to `CollectionItem` model scoped to the `logo` media collection via `performOnCollections('logo')`.

### Phase 4 — Debug generator button

- Added `seedWidgetCollections()` method to `DashboardDebugGeneratorWidget` calling `CarouselDemoSeeder`, `ChartDemoSeeder`, and `LogoGardenDemoSeeder`.
- Added "Seed widget collections" button (gray, independent of record type dropdown) to the debug generator blade template.

### Additional work

- Copied sample logo images from Windows mount (`/mnt/e/`) to `resources/sample-images/logos/` (9 files).
- Rebuilt Vite bundle to include `FreeMode` Swiper module.
- Wrote session 127 prompt: Board Members & Three Buckets Widgets.

## Production deployment notes

- Run `php artisan db:seed --class=WidgetTypeSeeder` to register the `logo_garden` widget type.
- Run `php artisan build:public` to compile the new SCSS into the public widget bundle.
- Run Vite build (`npm run build`) to include the `FreeMode` Swiper module in the admin/public JS bundle.
- Optionally run `php artisan db:seed --class=LogoGardenDemoSeeder` to create demo collection data.
