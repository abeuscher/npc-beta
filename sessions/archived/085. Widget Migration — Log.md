**Date:** 2026-03-28

## What was built

### Phase 1 — PageContext update

- `PageContext::event()` and `PageContext::product()` updated to accept `?string $slug`, returning `null` immediately for null input so templates can safely pass `$config['slug'] ?? null`
- `PageContext::product()` gains `->with('prices')` eager loading

### Phase 2 — Five broken widget templates migrated

- `event-description`, `event-registration`: `@php $event = $pageContext->event($config['event_slug'] ?? null); @endphp` prepended
- `events-listing`: `$events` now populated via `$pageContext->collection('events', $limit)` with the same int-cast limit pattern as `blog-listing`
- `product-display`: `@php $product = $pageContext->product($config['product_slug'] ?? null); @endphp` prepended

### Phase 3 — Seeder update, reseed, and cleanup

- `events_listing` `collections` cleared to `[]` in `WidgetTypeSeeder`
- Seeder rerun via Docker
- Three orphaned blade files deleted: `rich-text.blade.php`, `blog-roll.blade.php`, `collection-list.blade.php`

### Phase 4 — Widget type `default_open` flag

- Migration: `default_open boolean default false` added to `widget_types`
- `WidgetType` model: `default_open` added to `$fillable`, cast as boolean
- `PageBuilderBlock::loadBlock()`: `widget_type_default_open` added to the `$block` array
- Block view: Alpine `open` initialiser reads from `$block['widget_type_default_open']` instead of hardcoded `false`
- `WidgetTypeResource` form: Toggle field "Start open in page builder" added
- `docs/schema.md` updated

### Addendum — Mid-session additions

- `text_block` widget type: `default_open` set to `true` in `WidgetTypeSeeder`; reseeded
- Default post widgets: `CreatePost::afterCreate()` seeds a `text_block` (label: Content, sort 1) and `blog_pager` (label: Post Pager, sort 2) on every new post created via the admin
- `event_description` + `event_dates` merged: structured date, time, and location (is_in_person / is_virtual / city + state) computed in `@php`, output in `<p class="event-date">` with a `<time>` element; event title rendered above as `<h1 class="event-title">`; description below; `event_dates` widget type removed from seeder (existing DB records and their `page_widgets` deleted on reseed); `event-dates.blade.php` deleted; `EventTest` updated accordingly
- Widget picker modal: grid changed from 2 to 3 columns; inner panel changed from `w-full` to `container mx-auto`
