## What was built

### Phase 1 — Design System Editor page framework

- New Filament page `DesignSystemPage.php` under Tools navigation group
- Tabbed layout: Buttons (functional), Text Styles (placeholder)
- Help doc stub at `resources/docs/design-system.md` with route registration
- Breadcrumbs and `manage_cms_settings` permission gate

### Phase 2 — Button variant editor

- Five collapsible variant sections (primary, secondary, text, destructive, link)
- Eight controls per variant: border-radius (category picker), background color, text color, border color, border width, hover behavior (darken/lighten/opacity), font weight, text transform
- JSON storage on `button_styles` site setting (type: json, group: design)
- Alpine.js live preview strips showing two button sizes per variant
- Hex color validation on save — invalid values rejected to null

### Phase 3 — Icon buttons & form-append buttons

- Icon buttons section: icon size (match/larger/1.5x), placement (left/right/icon-only), mobile collapse toggle
- Form-append buttons section: default variant picker

### Phase 4 — SCSS compilation wiring

- Rewrote `_buttons.scss` to use CSS custom properties with hardcoded fallbacks matching previous styles
- Added base rules: nowrap/ellipsis overflow, focus-visible ring, disabled state, size modifiers (`.btn--sm`, `.btn--lg`), wrap/block-mobile modifiers
- Added icon button classes (`.btn--icon-left`, `.btn--icon-right`, `.btn--icon-only`, `.btn--icon-collapse-mobile`)
- Added form-append `.input-group` pattern
- Hover behaviors driven by `data-hover` attribute (opacity, darken via brightness filter, lighten)
- `AssetBuildService::generateButtonOverrideCss()` reads `button_styles` from DB at build time and injects `:root` custom properties into the combined SCSS — no file I/O needed
- Updated button component (`widgets/components/buttons.blade.php`): auto external-link icon with SVG for outbound URLs, `target="_blank"` with `rel="noopener noreferrer"`, subdomain-aware detection
- Save triggers `AssetBuildService::build()` with success/warning notification
- Button label changed to "Rebuild CSS Bundle" with arrow-path icon
