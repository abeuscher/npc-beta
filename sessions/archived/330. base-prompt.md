## Process rules for every session

Base-prompt process rules apply in full. Session-specific notes:

- **These are stylesheets, not irreversible commitments — drop the gate theater.** Breakpoint values, a spacing value, whether `@layer` ships: if the owner changes their mind in a month, it is a number edit + rebuild. Pick sane defaults, say what you picked, move on. The *only* genuine pause is the **post-audit alignment** (Phase 1 → 2) and the **visual check** ("does it still look right" — the one thing a test can't answer). Do not manufacture "expensive to reverse" framing around CSS.
- **Do not invent systems nothing needs.** There is no spacing scale in the code today — only `$gutter: 1.5rem`. Do **not** propose building an `--np-space-*` scale unless the audit finds a real widget that needs more than the one value. The default fix for a leak is the lightest one (widget carries its own value, or reads a single published token equal to the existing value), not a new vocabulary. Same for type tokens.
- **Phase 1 writes no application code.** The audit is findings + recommendations only. The fix happens in Phase 2, after alignment.
- **Audit surfaces inclusively.** Report every leak / specificity hazard / boundary crossing the audit finds, by matrix. The "won't-fix" call comes from the alignment walkthrough with the owner, **not** from filtering during the audit.
- **Behavior-preserving is the bar for Phase 2.** Every fix must leave the rendered output identical. The leak-consumption test proves the boundary is clean; a visual pass proves nothing moved. If a fix would change appearance, it is out of scope — it belongs to the later mobile-appearance design pass, not here.
- **Front-end builds:** widget SCSS / `widget_types.assets` / new widget asset → `docker compose exec app php artisan build:public` (regenerates `public/build/widgets/manifest.json`). `resources/scss/**` (including `_variables.scss`, `_layout.scss`, `_base.scss`) → `npm run build`. When a widget's runtime output could have changed, run `build:public` — a stale manifest silently breaks consumers.
