## Per-band decisions worth recording

- **Band 2 widget configuration approach: verbatim helper output.** `PricingChartDefinition::marketingSiteTiers()` was inspected end-to-end against the pricing-page spec before authoring `pricing.json`; no drift detected. Helper used as-is via PHP shim that generated the JSON literal — no per-instance overrides.
- **Band 3 grid shape: `columns: 4, grid_template_columns: "1fr 1fr"` over `columns: 2` with stacked widgets.** The session prompt's "default: standard `columns: 2, ...` layout container with four Text widgets, one per cell" was ambiguous between two structurally different shapes (4-slot single-widget-per-slot CSS-wrapped vs 2-slot two-widgets-stacked-per-slot). Chose the 4-slot variant on G20-avoidance grounds; the alternate would have re-triggered the global `.widget--text_block { height: 100% }` collision that About Band 2 burned through. Noted in gap-report under G20.
- **Band 5 anatomy compromise.** The spec called for primary CTA → small question text → secondary text-link as a vertical sequence. Text widget renders `ctas` as a single CTA row, not split into per-line groupings. Single-widget approach with the question paragraph in content + both CTAs in the array preserves the routes and the `secondary-dark` variant but loses the spec's "secondary as a text-link below" anatomy. Documented in gap-report; surface for user-judgment review if visually unclear.
- **Image placeholder picks.** Both placeholders match the spec's image-naming convention. Portrait choice (JORGE_LUIS_DEL_HIERRO) deliberately distinct from Home's Michael Brun (modern composed portrait, dark hair) and About's FDR (mid-century color portrait) per the spec's "third distinct founder portrait" directive. Band 4 supporting image (Corfu) reads as quiet/contemplative — appropriate for "principles" content per the spec's open guidance.
- **Body content matches `copy.md` only in spirit.** The pricing-page spec overrides `copy.md`'s outdated band structure for the Pricing page per its Authority section. The spec's bands (hero / comparison / à la carte / what-I-won't-build / final CTA) don't map 1:1 to `copy.md`'s Pricing bands (hero / what's-included / founder pricing / à la carte / what's-not-included / final CTA). Spec-canonical structure wins.
