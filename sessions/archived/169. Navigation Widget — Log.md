### Phase 3 ‚Äî Widget Template & Markup

- Rewrote `resources/views/widgets/nav.blade.php` with:
  - Eager-loaded 3-level NavigationItem tree (parent ‚Üí child ‚Üí grandchild).
  - Semantic `<nav>` with ARIA: `role="menubar"`, `role="menu"`, `role="menuitem"`, `aria-haspopup`, `aria-expanded`, `aria-current="page"`, `aria-controls`.
  - Branding slot (logo image, icon image, text) rendering based on `branding_type`.
  - Mustache token replacement for parent/child item templates with `e()` escaping.
  - Dropdown panel inline styles composed from border config + `GradientComposer` output.
  - Alpine.js interactivity: hover-open with 150ms close delay, focus-open for keyboard, arrow-down opens dropdown and moves focus, escape-to-close.
  - Mobile hamburger button with ARIA, Alpine `x-collapse` for nested sub-menus, chevron rotation.
  - Per-instance `<style>` block for the configurable mobile breakpoint media queries.
  - Link color CSS custom properties (`--nav-link-color`, `--nav-hover-color`, `--nav-drop-link-color`, `--nav-drop-hover-color`) set inline.
  - Empty state renders nothing when no menu is selected or no items exist.
  - Alignment parsing handles the single-word `center` case.
  - Gradient type-safety: only composes when config value is an array.

  - Mobile menu styles with nested indentation and chevron rotation.
  - `min-height: 5rem` on both `.widget-nav` and `.widget-logo`.

## Production deployment notes

After deploy, run:

```
docker exec nonprofitcrm_app php artisan db:seed --class=WidgetTypeSeeder --force
docker exec nonprofitcrm_app php artisan db:seed --class=SystemPageSeeder --force
docker exec nonprofitcrm_app php artisan build:public
```

The first re-seeds the nav widget type with the new config schema (existing nav widget instances keep their config; new installs pick up the new defaults). The second populates the `_header` and `_footer` chrome pages with default widgets if they are empty (idempotent). The third rebuilds the public widget asset bundle to pick up the new `_navigation.scss`.
