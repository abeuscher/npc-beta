This is the substantive, full-scope Phase 2 session in the Fleet Manager Agent track. It lands the greenfield CRM-side backup mechanism (none exists today), wires it to a DigitalOcean Spaces bucket (one bucket per install), records each successful backup's timestamp to a known location, and flips `HealthController::checkLastBackupAt()` from the v1.1.0 hardcoded `unknown` emission to a threshold-driven status read against that timestamp. The session sets up the operator's own install's bucket as the working example; documentation in `docs/app-reference.md` covers the per-install procedure for future installs.
