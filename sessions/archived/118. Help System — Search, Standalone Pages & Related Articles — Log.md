## What was built

### Phase 1 — Category taxonomy

- Added `category` string column (nullable) to `help_articles` table via migration.
- Updated `HelpArticle` model with `category` in `$fillable`.
- Updated `help:sync` command to read `category` from markdown frontmatter, validate against allowed values (crm, cms, finance, tools, settings, general), and warn on missing or invalid categories.
- Added `category:` frontmatter to all 44 markdown files in `resources/docs/` per the specified mapping.
- Updated `docs/schema/help_articles.md`.

### Phase 2 — Remove dashboard help widgets

- Deleted `DashboardHelpPlaceholderWidget` and `DashboardTaxonomyWidget` (PHP classes and blade views).

### Phase 3 — Help search in the top bar

- Created `App\Livewire\HelpSearch` component with ranked search: title > tags > description > content. Debounced 300ms, max 8 results, category badges.
- Registered via `GLOBAL_SEARCH_BEFORE` render hook in `AdminPanelProvider`.
- Blade template with Alpine.js keyboard navigation (arrow keys, enter, escape).
- Full CSS for search input, dropdown, result items, and category badges (light + dark themes) in `public/css/admin.css`.

### Phase 4 — Standalone help pages

- Created `HelpArticlePage` Filament page at `/admin/help/{slug}` — renders article markdown, category badge, metadata, related articles sidebar (tag overlap query, up to 6, alphabetised by title).
- Created `HelpIndexPage` at `/admin/help` — centred search field with "How can we help?" heading.
- Created `HelpCategoryPage` at `/admin/help/category/{category}` — flat list of articles with title links and descriptions.
- Breadcrumbs: Help > {Category} > {Article Title} with clickable links.
- Added "Open in full page →" link to the existing slide-over footer.
- All help pages require authentication, do not appear in left navigation.

### Session planning — Widget sessions 119–132

- Wrote 14 session prompts and base prompts for the widget system expansion:
  - 119: Widget picker infrastructure (category, allowed_page_types, grouped sections, search filter)
  - 120–132: 13 individual widget sessions (Hero, Full Size Hero, Video/Img Background, Logo Garden, Board Members, Three Buckets, Alternating Panels, Product Carousel, Event Map, Headline Patterns, CTA Patterns, Social Sharing, Modal)
- Updated `sessions/session-outlines.md` with all new sessions under "Widget System — Beta 1 Scope".
