## What was built

### Phase 1 — Board Members widget type registration

- Added `board_members` entry to `WidgetTypeSeeder` with `['members']` collection slot, 19 config fields covering collection field mapping, image shape, colour group, column count, alignment, and border radius.

### Phase 2 — Board Members Blade template and SCSS

- Created `resources/views/widgets/board-members.blade.php` with flexbox grid of `<article>` cards.
- Images rendered as `<img>` inside a clipped container using `object-fit: cover` and `object-position: top center` for portrait cropping. Circle mode uses `clip-path: circle(50%)`.
- Aspect ratio via padding-bottom trick (computed from config ratio string).
- Layout controlled entirely by SCSS classes (`board-members--cols-N`, `board-members--align-left/center/right`). Only truly dynamic values (hex colours, border-radius px, aspect-ratio padding) use inline custom properties.
- Responsive: 2 columns at `$bp-md`, single column at `$bp-sm`.
- Social links section with LinkedIn and GitHub SVG icon partials and optional extra URL.

### Phase 3 — Three Buckets widget type registration

- Added `three_buckets` entry to `WidgetTypeSeeder` with 3× heading/body/buttons groups, alignment controls, and custom gap.
- Button fields reuse the same `buttons` type and field structure as the hero widget.

### Phase 4 — Three Buckets Blade template and SCSS

- Created `resources/views/widgets/three-buckets.blade.php` with `@for` loop over three buckets, each rendering heading, rich text body, and shared buttons partial.
- Alignment via SCSS modifier classes (`three-buckets--headings-left/center/right`, `three-buckets--body-left/center/right`). No inline styles for alignment.
- Added `$gutter: 1.5rem` to `resources/scss/_variables.scss`.
- SCSS uses `justify-content: space-between` with `var(--bucket-gap, $gutter)`. Stacks at `$bp-md`.

### Phase 5 — Demo collection seeder and debug generator

- Created `BoardMembersDemoSeeder` with 6 board members using `fake()` for names and bios.
- Portrait images from `resources/sample-images/portraits/` attached via Spatie media library.
- Added `seedWidgetCollections()` method to `DashboardDebugGeneratorWidget` running all three demo seeders (carousel, chart, board members).
- Added "Seed widget collections" button to the debug generator UI.

### Additional
