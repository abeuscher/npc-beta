## What was built

### Phase 1 — "Most Used" Widget Category

- Added `most_used` to `WidgetTypeResource::CATEGORY_OPTIONS` as a standard category. Widgets are assigned to it via the existing category checkbox list in Widget Manager — no automatic tracking, no special logic.
- Updated the widget picker modal to default to `most_used` as the active category when any widgets have that category assigned. Falls back to "All" otherwise.
- Changed the widget picker modal from `max-height: 90vh` to `height: 90vh` for persistent sizing.

### Phase 2 — Text Drop Shadow

- Added a "Drop Shadow" checkbox toggle to `TextPanel.vue` in the appearance inspector, below the existing text color picker.
- Added text-shadow emission in `AppearanceStyleComposer::compose()` — emits `text-shadow:0 1px 3px rgba(0,0,0,0.6)` when `text.shadow` is truthy in `appearance_config`.
- Updated `docs/schema/page_widgets.md` with the new `text.shadow` key in the appearance config shape.

### Bug fix — Quill Dropdown Clipping

- Fixed Quill toolbar dropdowns (heading picker, etc.) being clipped at the bottom of the editor window. Removed `overflow: hidden` from `.inspector-richtext` in `RichTextField.vue` and replaced with `position: relative`.
