**Date:** 2026-04-06

## What was built

### Phase 1 — Tab infrastructure

- Added Alpine-powered tab bar to the inspector panel with two tabs: Content and Appearance
- Tab state managed via `x-data="{ activeTab: 'content' }"` — pure Alpine toggle, no server round-trip
- Active/inactive styling with `x-bind:class` for opacity, background, and border transitions
- Tab content areas use `x-show` with `x-cloak` for instant switching

### Phase 2 — Content / Appearance split

- Config fields routed into tabs by `group` key in `config_schema` (defaults to `content` when group is missing)
- Query Settings section (collection handle, limit, direction, order, tag filters) moved to Appearance tab
- Spacing panel (padding, margin, full-width toggle) moved to Appearance tab
- Advanced/carousel accordion renders only in the tab that contains advanced fields for that group
- Empty-tab placeholder ("No content settings for this widget") shown when a widget has no content-group fields
- Appearance tab always shows spacing controls even when no appearance-group config fields exist

### Phase 3 — Tab styling

- Active tab: `bg-white dark:bg-gray-800`, full opacity, bottom border matches panel background (connected look)
- Inactive tab: transparent background, `opacity-60`, hover to `opacity-80`
- Tab text: `text-xs font-semibold uppercase tracking-wide` matching existing inspector section headers
- Rounded top corners (`rounded-t-md`) on each tab
- Tab bar background: `bg-gray-50 dark:bg-gray-900` with side borders for visual separation
- `transition-all duration-150` for smooth state changes
- Full dark mode support throughout

### Additional — Apply Changes button relocation

- Moved Apply Changes button from inside the inspector scroll area to a sticky bottom bar
- `sticky bottom-0` keeps the button pinned at the bottom of the inspector's scroll container at all times
- Subtle backdrop blur (`bg-white/95 backdrop-blur-sm`) and top border for visual separation from scrolling content
- Block label remains outside tabs, always visible at the top
