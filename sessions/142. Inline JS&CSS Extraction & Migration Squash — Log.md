### Bug fixes

- **Anchor underline** ‚Äî removed `text-decoration: underline` from `.np-site a` in `_base.scss` (added accidentally when the `.np-site` namespace was introduced)
- **Viewport switcher** ‚Äî `this.$el` in Alpine.data() resolves dynamically based on calling context; when called from a button click handler, it returned the 24px-wide button instead of the pane div. Fixed by capturing the root element as `_pane` in `init()`.
- **Widget reinit errors** ‚Äî wrapped `Alpine.initTree()` calls in `reinitWidgetAlpine` with try/catch so widget errors (e.g. bar chart missing `$refs.chartCfg`) cannot break the preview manager's async chain.
- **Widget picker grid** ‚Äî added `resources/views/components/` to Filament Tailwind content scan; classes like `grid-cols-6` were being purged because the widget picker modal (extracted to a Blade component in session 141) wasn't in the scan path.
- **Seeder** ‚Äî About and Contact pages now seed as published (were left as draft from early publish-mode testing).
