### Phase 3 — Apply picks

#### `/1` — drop bootstrap widgets duplicate (B1 / 208-deferred B4) (`0d79949`)

`PageBuilder::getBootstrapData()` shipped both `widgets` (a derived subset of `items` keeping only `type=widget` entries) and `items`. The Vue `BootstrapData` interface declares only `items`; the editor store calls `populateFromItems(data.items ?? [])` and never reads `data.widgets`. Three-line removal in `app/Livewire/PageBuilder.php`. Resolves the deferred B4 pick from session 208 — the Vue audit that 208 deferred is now done.
