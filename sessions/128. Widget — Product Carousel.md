# 128. Widget ŌĆö Product Carousel

Add a `product_carousel` widget type: a Swiper-based carousel that displays published products with image, name, price, and a link to the product checkout page. Uses the existing products data source in `WidgetDataResolver`.

---

## Design decisions (resolved before starting)

- Product data comes from `WidgetDataResolver` using the `products` source type ŌĆö the same pipeline the existing product-related features use.
- This widget uses Swiper (already loaded for the carousel widget). Same Swiper config pattern as the existing `carousel` widget.
- Categorised as `commerce` with `allowed_page_types: null`.
- Each slide shows the product's featured image, name, price, and a "View" or "Buy" link.

| Key | Label | Type | Default | Notes |
|---|---|---|---|---|

| pagination | Show Dots | toggle | true | |
| cta_text | Button Text | text | 'View' | Text on each product's link |

### Template

Swiper carousel. Each slide renders a product card: featured image (from `_media`), product name, price (formatted), and a CTA link to the product's public page. Follows the same Swiper initialisation pattern as the existing carousel widget. Responsive ŌĆö fewer slides on smaller viewports.
