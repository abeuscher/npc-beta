## What was built

### Phase 1 ‚Äî Scheme model + the single shared resolver

- **`App\Services\TemplateAppearanceResolver`** ‚Äî the one shared resolver: a `Template` ‚Üí the request-time inline `--np-color-*` content-region override string. Fixed scheme set (`default` + `inverse`); `default` = empty delta (identity); `inverse` = a contrast-vetted dark assignment over **only** the 7 content-region tokens (never `brand`, never chrome tokens ‚Äî compose-not-bleed by construction). Concrete-values: null/unset/unknown scheme ‚Üí Default ‚Üí empty (a fresh install renders byte-identical with no row). Structured so a power-user named-scheme layer is an additive follow-up; the author only ever *selects*.
- **Wired into both call sites with no divergent branch:** the public layout (`resources/views/layouts/public.blade.php`) applies it to `<main>` (content region only); the page-builder preview (`PageBuilder::previewContentSchemeVars()` ‚Üí `livewire/page-builder.blade.php`) applies the identical string scoped to `.page-builder .widget-preview-scope`. Request-time inline, never compiled into the bundle ‚Äî outside the 296 guard by design.
