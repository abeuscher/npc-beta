# 120. Widget — Hero

Add a `hero` widget type: a full-width banner with heading, subheading, background image, optional CTA button, and configurable text alignment and overlay opacity. This is the standard marketing hero — a large image with text on top.

---

## Design decisions (resolved before starting)

- The hero uses the standard widget contract: `config_schema` fields, server-rendered Blade template, `configMedia` for the background image.
- No new infrastructure — this is a pure widget addition using existing systems.
- The background image uses the existing Spatie media pipeline with responsive breakpoints and webp conversion.
- Text overlay uses a semi-transparent dark gradient. Opacity is configurable.
- The widget is categorised as `content` with `allowed_page_types: null` (all pages).

## Implementation

### Seeder entry

Add to `WidgetTypeSeeder`:

| Field | Value |
|---|---|
| handle | `hero` |
| label | `Hero` |
| category | `content` |
| allowed_page_types | null |
| render_mode | server |
| default_open | false |
| collections | [] |

### Config schema

| Key | Label | Type | Default | Notes |
|---|---|---|---|---|
| heading | Heading | text | '' | |
| subheading | Subheading | textarea | '' | |
| background_image | Background Image | image | null | |
| cta_text | Button Text | text | '' | Leave empty to hide button |
| cta_url | Button URL | url | '' | |
| text_align | Text Alignment | select | 'center' | Options: left, center, right |
| overlay_opacity | Overlay Opacity | number | 50 | 0–100, rendered as percentage |
| min_height | Minimum Height | select | '24rem' | Options: 16rem, 24rem, 32rem, 40rem |

### Template

Server-rendered Blade template. The background image is set via inline `background-image` style using the resolved media URL. The overlay is a positioned div with configurable opacity. Text and CTA button are centred (or aligned per config) on top.

Use Tailwind utility classes consistent with existing widget templates. The CTA button should use the site's primary colour via CSS custom property.

### CSS

Scoped to `.widget--hero`. Minimal — leverage Tailwind. Any custom styles go in the widget's `css` field.
