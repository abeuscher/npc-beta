We are about to begin a new session: **309. Content Import/Export — Opt-in Theme + Interactive Import Prompts.**

The work is to make the seam explicit: opt flags on the exporter, a pre-flight `analyze()` method on the importer, a two-step UI on the import form, and a flipped default (`merge_design = false`) so old bundles don't silently overwrite theme.
