## Open questions to resolve at session start

- **Image placeholder picks across the five cells.** Sample image library has 11 portraits, 9 still-photos, 10 product-photos, 14 logos. The spec's four named cells call for portraits or still-photos. Pick fits that don't try to anticipate the eventual real-photo aesthetic (staged-stock-photo with the founder as subject); the brief is explicit on this point.
- **Band 1 hero composition.** The spec's left cell carries headline + body + CTAs. The spec allows two execution paths: single Hero widget in the left cell (Hero carries all three), or a Text widget with the new CTA capability in the left cell. With G2 resolved, the Text-widget path is available — but Hero remains the higher-fidelity carrier for the hero band's full-bleed feel. Recommendation: Hero widget in the left cell, Image widget in the right. Confirm before serializing Band 1.
- **Bands 4 / 5 / 6 / 8 CTA carrier choice.** With G2 resolved, these bands can use Text widget + new CTAs (lighter; honors the spec's "Text widget" phrasing). Alternative: Hero widget for the visual heft. Make the call once for consistency across bands rather than per-band.
