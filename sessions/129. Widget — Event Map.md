# 128. Widget — Event Map

Add an `event_map` widget type: an embedded Google Maps iframe showing an event or organisation location. The admin pastes a Google Maps share URL and the widget converts it to an embed.

---

## Design decisions (resolved before starting)

- This widget accepts a Google Maps URL (share link or place URL) and renders it as an iframe embed. No API key required for basic iframe embeds.
- The conversion from share URL to embed URL follows the same pattern as `video_embed` — parse the input URL and transform it into the correct iframe `src`.
- Categorised as `media` with `allowed_page_types: null`. Despite the name "Event Map", this is a general-purpose map embed — useful on any page.
- No JavaScript mapping library needed. This is a simple iframe embed.

## Implementation

### Seeder entry

| Field | Value |
|---|---|
| handle | `event_map` |
| label | `Map Embed` |
| category | `media` |
| allowed_page_types | null |
| render_mode | server |

### Config schema

| Key | Label | Type | Default | Notes |
|---|---|---|---|---|
| map_url | Google Maps URL | url | '' | Paste a Google Maps share or place URL |
| height | Map Height | select | '24rem' | Options: 16rem, 24rem, 32rem, 40rem |
| heading | Heading | text | '' | Optional heading above the map |
| caption | Caption | text | '' | Optional text below the map (e.g. address) |

### Template

Parse the `map_url` config value. Extract the location query or place ID and construct a Google Maps embed iframe URL. Render the iframe with the configured height, full width, no border, `loading="lazy"`. Show heading above and caption below if provided.

**URL parsing rules:**
- `https://www.google.com/maps/place/...` → extract coordinates or place query, construct `https://www.google.com/maps/embed?pb=...` or use `https://maps.google.com/maps?q=...&output=embed`
- `https://goo.gl/maps/...` or `https://maps.app.goo.gl/...` — short links. These can be used directly in an iframe `src` with `output=embed` appended, or the widget can display a helpful error message asking for the full URL.
- If the URL doesn't match known patterns, render a fallback message ("Paste a Google Maps URL to display a map").

Investigate the simplest reliable conversion. If straightforward conversion from share URLs works, implement it. If Google Maps URL patterns are too variable, simplify to: accept any Google Maps URL and wrap it in an iframe using the `maps.google.com/maps?q=ENCODED_QUERY&output=embed` pattern with the URL-encoded search query extracted from the input.

### CSS

Responsive container. Map fills width, configurable height. Rounded corners consistent with other widget styles.
