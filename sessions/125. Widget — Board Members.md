# 125. Widget — Board Members

Add a `board_members` widget type: a people grid sourced from a custom collection, displaying photo, name, title, and optional bio. Suitable for board of directors, staff pages, or team sections.

---

## Design decisions (resolved before starting)

- People data comes from a custom collection via `WidgetDataResolver`. The admin creates a "Board Members" (or "Team", "Staff") collection with fields for photo, name, title, and bio, then points this widget at it.
- Standard collection-backed widget — same contract as carousel and logo garden.
- Categorised as `content` with `allowed_page_types: null`.
- Layout options: grid (cards) or list (horizontal rows). Both are config-driven variants in the same template.

## Implementation

### Seeder entry

| Field | Value |
|---|---|
| handle | `board_members` |
| label | `Board Members` |
| category | `content` |
| allowed_page_types | null |
| render_mode | server |
| collections | ["people"] |

### Config schema

| Key | Label | Type | Default | Notes |
|---|---|---|---|---|
| collection_handle | Collection | select | '' | Source collection |
| photo_field | Photo Field | text | 'photo' | Field key for person photo |
| name_field | Name Field | text | 'name' | Field key for person name |
| title_field | Title Field | text | 'title' | Field key for role/title |
| bio_field | Bio Field | text | 'bio' | Field key for short biography |
| layout | Layout | select | 'grid' | Options: grid, list |
| columns | Columns | number | 3 | Grid columns on desktop (1 on mobile, 2 on tablet) |
| heading | Heading | text | '' | Optional section heading |

### Template

**Grid layout:** Responsive card grid. Each card shows a circular/rounded photo, name (bold), title (muted), and truncated bio. Photo uses `object-fit: cover` with a fixed aspect ratio.

**List layout:** Horizontal rows with photo on the left, name/title/bio on the right. Photo smaller than grid variant.

Both layouts use Tailwind utilities. Photos come from `$collectionData` media fields via the standard pipeline.
